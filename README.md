# Exotic Pets Spring Java

**Exotic Pets Spring Java** – учебное backend-приложение на **Spring Boot**, реализующее REST API для управления экзотическими питомцами и пользовательскими профилями.

Проект демонстрирует базовые принципы построения серверного приложения на Java:
- слоистую архитектуру (Controller → Service → Repository),
- работу с REST-эндпоинтами,
- передачу данных через DTO,
- обработку ошибок и валидацию входных данных.

---

Приложение моделирует простую предметную область:

- **Profile** — пользовательский профиль с логином и паролем:
  - уникальный идентификатор (`UUID`);
  - логин (`login`);
  - пароль (`password`);

- **Pet** — питомец, принадлежащий конкретному профилю
  - уникальный идентификатор (`UUID`);
  - имя (`name`);
  - характеристику силы (`strength`);
  - связан с владельцем через `profileId`.
